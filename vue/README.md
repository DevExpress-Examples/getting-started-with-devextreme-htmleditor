# DevExtreme Vue HtmlEditor - Getting Started 

This repository stores the code examples of the Vue HtmlEditor component for the [Getting Started with HtmlEditor](https://js.devexpress.com/Documentation/Guide/UI_Components/HtmlEditor/Getting_Started_with_HtmlEditor/) tutorial. The HtmlEditor is a client-side WYSIWYG editor that allows users to format textual and visual content and output it as HTML or Markdown. This tutorial shows how to add the HtmlEditor to a page, bind it to data, and configure its core features.
